# Small View Model
Change the position, rotation and scale of your view model!

This mod uses the [ModMenu Mod](https://github.com/TerraformersMC/ModMenu) to change the settings and [Lattice](https://github.com/moulberry/lattice) for the config.

## Versions
Currently, the mod is only supported for Fabric 1.21.8.
